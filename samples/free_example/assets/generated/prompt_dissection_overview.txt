[1단 — PROBLEM (좌측)]
헤더: "PROBLEM"
시각: 32-layer LLM 박스가 세로로 그려져 있고 모든 토큰("a", "the", "dog", "sunglasses")이 마지막 32층까지 통과하는 화살표가 굵게. 그 위에 빨간 라벨 "Every token, every layer = WASTE". 하단에 작은 그래프 — y=accuracy, x=layers, V자 함몰(중간 레이어)이 빨간 박스로 강조, "Mid-crisis" 라벨. 그 옆 yellow 박스 "Overthinking".
서브텍스트(작게): "VLM decoders always run all N layers — wasteful AND prone to mid-layer dips"

[2단 — KEY OBSERVATIONS (좌중간)]
헤더: "KEY OBSERVATIONS"
세로로 stack된 4개 아이콘 박스:
- V자 곡선 + "Mid-crisis is systematic (10-15 of 32 layers)"
- 두 hidden state 벡터가 같은 분류기로 들어가는 그림 + "If ET output ≈ final → reuse C_N"
- 두 곡선(cosine: flat, adversarial: rising) + "Adversarial > cosine for generalization"
- 빨간 X(mode collapse) + 녹색 체크(hard label save) + "Hard labels prevent mode collapse"

[3단 — METHOD (중앙, 가장 크게)]
헤더: "FREE PIPELINE"
중앙에 큰 LLM 박스, 안에 Layer 1, Layer 2, ..., Layer N(녹색 자물쇠 ❄으로 모두 동결 표시). Layer 1, Layer N/2, Layer N에 exit 부착:
- 각 exit = ET(녹색, "trainable") + FC(분홍, "discriminator, train-only") + EC(노란, "= C_N frozen ❄")
- ET가 LM layer 복제임을 표시(같은 색·모양)
좌측 아래: input image → ViT-g → Q-Former → LLM 흐름.
우측: GAN 학습 다이어그램 — ET 출력 vs Final layer 출력 → FC → "real or fake?"
하단 핵심 수식: "L_i = L^gen_i + L^CE_i (or L^KL_i)"
"Inference: S_i = max_v C_N(E_i(h^i_t)) ≥ τ → exit here"

[4단 — WHAT'S NEW (중우측)]
헤더: "WHAT'S NEW"
좌우 비교 박스 5개:
- "DeeBLIP: new classifier per exit, label-hungry" (빨간 X)
- "PABEE-BLIP: patience-based, no representation alignment" (빨간 X)
- "LeeBLIP: KD only, no adversarial" (빨간 X)
- "MuE: cosine similarity = weak generalization" (빨간 X)
- "FREE: GAN-aligned ET + frozen C_N reuse + 3 setups" (녹색 체크, 강조)
